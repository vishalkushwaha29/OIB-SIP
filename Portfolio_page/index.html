Blog

My Journey to Becoming a Web Developer

When I first started learning web development, I barely knew how websites actually worked. But with consistency and curiosity, I started exploring HTML, CSS, and JavaScript — and eventually moved on to React, Node.js, and Express. The turning point came when I joined the Apna College SIGMA Batch, where I learned full-stack development and built real-world projects like Wanderlust and Weather App. Each project helped me connect theory with real code. I’ve realized one thing — it’s not about learning everything fast, it’s about learning one thing deeply and building something real with it.

Building My First Full Stack Project — Wanderlust

Wanderlust was my first big project as a full-stack developer, inspired by Airbnb. I used the MERN stack (MongoDB, Express, React, Node) to create a platform where users can add, edit, and explore travel listings.The biggest challenge was setting up authentication and session management, but I learned how to handle it using Passport.js. I also improved my UI design using Bootstrap to make the website responsive and clean.This project not only boosted my technical skills but also taught me patience — debugging Node errors at 2 AM is now my superpower.

How I Built a Weather App

Creating a weather app helped me understand how APIs work. I used the OpenWeather API to fetch live weather data and Bootstrap to make the design clean and responsive.The most interesting part was making the UI change dynamically based on the weather — sunny, rainy, cloudy — all with simple JavaScript logic.Through this project, I learned how frontend and APIs connect together to create real-time user experiences.

Why I Love Building with React

When I first started learning React.js, it felt confusing — JSX, components, props, states — everything seemed complex. But once I understood the flow, it completely changed how I build websites.I love how React makes UI development modular and efficient. Instead of rewriting the same code, I can reuse components anywhere. Using React Hooks like useState and useEffect helped me manage logic cleanly and handle API calls easily. Today, I use React to build fast, dynamic, and responsive web apps — and it’s now my favorite frontend library for creating real-world projects.
